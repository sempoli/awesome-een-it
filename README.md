# Awesome For Beginners  [![Stars](https://img.shields.io/github/stars/yourname/awesome-for-beginners?style=social)](https://github.com/yourname/awesome-for-beginners) [![Forks](https://img.shields.io/github/forks/yourname/awesome-for-beginners?style=social)](https://github.com/yourname/awesome-for-beginners)

> A curated list of beginner-friendly open-source projects looking for contributions.

Perfect place to start contributing to open source! All projects here have the label **good first issue** / **help wanted**.

## JavaScript / TypeScript
- [freeCodeCamp/freeCodeCamp](https://github.com/freeCodeCamp/freeCodeCamp) – Learn to code for free
- [public-apis/public-apis](https://github.com/public-apis/public-apis) – Collective list of free APIs

## Python
- [Zappa](https://github.com/Miserlou/Zappa) – Serverless Python
- [httpie](https://github.com/httpie/httpie) – Modern cURL replacement

## Go
- [traefik](https://github.com/traefik/traefik) – Modern reverse proxy
- [fzf](https://github.com/junegunn/fzf) – Command-line fuzzy finder

## Rust
- [rust-lang/rust](https://github.com/rust-lang/rust) – The Rust language itself
- [denoland/deno](https://github.com/denoland/deno)

## Contributing
Found another beginner-friendly project? Open a Pull Request!

1. Fork the repo
2. Add the project under the correct category
3. Keep the format: `- [repo-name](link) – short description`
4. Open PR ❤️
